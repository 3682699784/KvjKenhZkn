# Python医药信息管理系统简介

## 核心功能
- 信息查询
- 信息录入
- 信息删除
- 信息修改
- 数据报表
- 信息浏览

## 适用场景
- 医疗机构信息管理
- 药品库存管理
- 顾客资料管理
- 经办人信息管理

## 技术特点
- 采用Python语言和Django框架进行开发
- 使用HTML、CSS、JavaScript和jQuery构建前端界面
- 后端数据库使用MySQL
- 支持Python3.0以上版本和MySQL8.0环境
- 可通过Pycharm和navicat进行开发和数据库管理
